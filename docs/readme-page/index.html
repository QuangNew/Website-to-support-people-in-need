ReliefConnect

A relief connection platform — supporting people in need

📖 About

ReliefConnect is a web application that connects people in need with organizations and individuals who want to help. The platform operates on a real-time interactive map, allowing users to send SOS signals, post supply information, and coordinate relief efforts during natural disasters, epidemics, or difficult circumstances.

✨ Key Features

🗺️

Interactive Map

Leaflet + OpenStreetMap with real-time SOS pings, supply points, danger zones, and PostGIS spatial indexing.

🔍

Smart Routing

OSRM integration with custom route deviation analysis, fallback alternative generation, and composite scoring.

💬

Real-time Messaging

SignalR WebSocket with read receipts, typing indicators, offline sync, and notification bell.

🤖

AI Assistant

Gemini AI chatbot for relief guidance, content moderation, and API key pool rotation.

🔐

Auth & Security

JWT + Cookie dual auth, Google OAuth, role-based access, email verification, and KYC workflow.

💰

Donation System

PayOS QR code payments, automatic webhook verification, and donation leaderboard.

🛡️

Admin Dashboard

User management, content moderation, audit logs, CSV exports, and soft-delete with restore.

🌍

Multilingual

Vietnamese + English i18n, dark/light mode, responsive desktop + mobile, glassmorphism UI.

🧮 Notable Algorithms

Spatial Query Optimization

PostGIS GIST spatial index enables O(log n) SOS radius searches.

Route Deviation Scoring

Custom composite score combining node overlap ratio, spatial deviation, and distance/duration penalties.

Viewport Ping Caching

Only fetches new pings when viewport exceeds cached bounds; merges results to prevent data loss.

Cross-origin Auth Strategy

Bearer header as primary auth, HttpOnly cookie fallback, query string for SignalR WebSocket.

🧮 Thuật toán nổi bật

Tối ưu truy vấn không gian

PostGIS GIST spatial index cho phép tìm kiếm SOS trong bán kính với O(log n).

Chấm điểm độ lệch tuyến đường

Hệ thống scoring tự phát triển kết hợp tỷ lệ trùng lặp node, độ lệch không gian, và phạt khoảng cách/thời gian.

Caching ping theo viewport

Chỉ fetch ping mới khi viewport vượt vùng cache; merge kết quả để tránh mất dữ liệu.

Chiến lược xác thực cross-origin

Bearer header làm auth chính, HttpOnly cookie dự phòng, query string cho SignalR WebSocket.

📖 概要

ReliefConnect は、支援を必要とする人々と支援を提供したい組織・個人をつなぐWebアプリケーションです。リアルタイム対話型マップを基盤とし、SOSシグナルの送信、物資情報の投稿、自然災害・疫病・困難な状況における救援活動の調整を可能にします。

🧮 注目のアルゴリズム

空間クエリ最適化

PostGIS GIST空間インデックスにより、SOS半径検索をO(log n)で実行。

ルート偏差スコアリング

ノード重複率、空間偏差、距離/時間ペナルティを組み合わせたカスタムスコアリングシステム。

ビューポートベースのピンキャッシング

ビューポートがキャッシュ範囲を超えた場合のみ新規ピンを取得し、データ損失を防止。

クロスオリジン認証戦略

Bearerヘッダーを主要認証とし、HttpOnly Cookieをフォールバック、SignalR WebSocket用にクエリストリング。
